# 테마 커스터마이징

`styles.css`를 수정해 브랜드 아이덴티티에 맞게 스타일을 바꿀 수 있습니다.

```css
:root {
  --accent: #1c7ed6;
  --background: #0b1324;
}
```

- 색상: `--accent`, `--background`, `--text` 값을 변경하세요.
- 폰트: `--font-title`, `--font-body`를 원하는 웹폰트로 교체할 수 있습니다.
- 카드 레이아웃: `.doc-card` 클래스를 참고해 블로그 글 목록을 꾸밀 수 있습니다.

작은 변경이라도 새로고침 후 확인하고, 어두운 배경/밝은 배경 모두에서 대비를 체크하세요.
